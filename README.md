# ant-mall

## 技术栈

### 前端技术栈
- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **Vue Router** - 路由管理
- **Axios** - HTTP客户端
- **Element Plus** - UI组件库
- **Pinia** - 状态管理

### 后端技术栈
- **Java 21** - 开发语言
- **Spring Boot 3.2.0** - 后端框架
- **MyBatis Plus** - ORM框架
- **MySQL** - 数据库
- **Knife4j** - API文档工具
- **Lombok** - Java代码简化库
- **Spring Validation** - 数据校验框架

## 项目结构
- `antmall-frontend/` - 前端项目
- `src/` - 后端项目
